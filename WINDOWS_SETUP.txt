==========================================

PROBLEM: Tesseract path error on Windows
SOLUTION: The app now auto-detects Tesseract installation

YOUR TESSERACT PATH: C:\Users\tx901465\AppData\Local\Programs\Tesseract-OCR

QUICK FIX:
1. Make sure tesseract.exe exists at: C:\Users\tx901465\AppData\Local\Programs\Tesseract-OCR\tesseract.exe

2. If file doesn't exist, download Tesseract from:
   https://github.com/UB-Mannheim/tesseract/wiki

   pip install -r requirements.txt

4. Run the app:
   python main.py
